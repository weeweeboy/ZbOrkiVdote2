Проект существует для того, чтобы закрыть мою личную нужду. У меня на компе мало оперативки, и при потной каточке в доту + запущенному дискорду, запуск браузера с большим количеством вкладок, крайне пагубно влияет на производительность компьютера!
Поэтому я и решил, что зайти в тг и написать имя героя + выбрать желаемую позицию будет гораздо проще для моуего компа. Данные по сборкам всегда актуальны и берутся с дота2протрекера. Удачи на полях сражения!
